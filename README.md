# JARVIS AI - React Native Android App

An Iron Man-inspired AI assistant mobile application built with React Native.

## Features

- Arc Reactor Animation - Animated spinning rings and pulsing core
- Voice Recognition - Real-time speech-to-text input
- Text-to-Speech - Natural voice responses
- Mobile HUD Interface - Cyberpunk-themed UI
- Terminal Display - Command logging and feedback
- Real-time Status - System monitoring and clock

## Installation

### Prerequisites
- Node.js >= 14
- Android SDK
- React Native CLI

### Setup

```bash
git clone https://github.com/miragegamingss-blip/Jarvis-AI.git
cd Jarvis-AI
npm install
```

### Run on Android

```bash
npm run android
```

## Commands

- "Hello Jarvis" - Greeting
- "Status" or "Report" - System status
- "What time is it" - Current time
- "Clear" - Clear terminal
- "Power" - Power status
- "Thanks" - Thank you response
